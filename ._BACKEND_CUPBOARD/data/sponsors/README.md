# Sponsor Data

Store sponsor metadata JSON files here.

## Naming convention

`<sponsor-slug>.json` — e.g. `palmerbet.json`, `kayo-sports.json`

## Schema

```json
{
  "sponsorId": "palmerbet",
  "name": "Palmerbet",
  "logoUrl": null,
  "website": "https://www.palmerbet.com.au",
  "tier": "gold",
  "events": ["ppv-ultimate-legends-apr-2026"]
}
```
