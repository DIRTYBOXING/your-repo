PRE-TEST SETUP
═══════════════════════════════════════════════════════════════════

  Equipment needed:
  [ ] 3-4 UWB anchors (DW3000 or equivalent), fully charged
  [ ] 2 CHUCKYA bracelets, charged >80%
  [ ] 1 Android phone with CHUCKYA app installed (debug APK)
  [ ] 1 laptop with Docker Compose stack running
  [ ] Tape measure or laser rangefinder
  [ ] Masking tape for anchor positions

  Services running on laptop:
  [ ] docker compose up -d  (all services healthy)
  [ ] Verify: curl http://localhost:8081/healthz → 200
  [ ] Control room open in browser (http://localhost:3000 or :80)

TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

  UWB range too short:
  - Raise anchor height; ensure line of sight
  - Check antenna orientation (vertical preferred)
  - Reduce interference from metal/water

  Distance readings jumpy:
  - Average 3-5 readings per test
  - Check for multipath reflections (glass, metal)
  - Firmware: increase ranging interval for stability

  Direction inaccurate:
  - Ensure bracelet orientation consistent between tests
  - AoA requires 2+ anchors; single anchor gives distance only

  No BLE notification:
  - Check phone BLE is ON and app has permissions
  - Verify bracelet is advertising (use nRF Connect app)
  - Re-pair if bond was lost
