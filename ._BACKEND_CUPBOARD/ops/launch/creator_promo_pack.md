# Creator Promo Pack

**Purpose**
Provide creators everything they need to promote the event quickly and compliantly: ready assets, caption options, promo code, UTM links, payment terms, and tracking instructions.

---

## Files to Deliver to Each Creator

| File                                            | Specs                                                       |
| ----------------------------------------------- | ----------------------------------------------------------- |
| `clip_15s_[eventslug]_[creator].mp4`            | Vertical 9:16, 1080×1920, H.264, 30 fps, max 15 s, 6–8 Mbps |
| `clip_30s_[eventslug]_[creator].mp4`            | Vertical 9:16, 1080×1920, H.264, 30 fps, max 30 s, 6–8 Mbps |
| `poster_1080x1350_[eventslug]_[creator].jpg`    | 1080×1350 px, sRGB, quality 85                              |
| `thumbnail_1080x1920_[eventslug]_[creator].jpg` | 1080×1920 px for stories/reels                              |
| `captions_[eventslug]_[creator].txt`            | Three caption options including disclosure                  |
| `promo_code_[creator].txt`                      | Unique code and landing URL with UTM                        |
| `invoice_template_[creator].docx`               | Blank invoice for creator to fill (if paid-fee chosen)      |

### Deliverable Checklist

- [ ] Assets uploaded to shared folder
- [ ] Promo code assigned and recorded in campaign spreadsheet
- [ ] Payment option confirmed: **Flat fee** or **Revenue share**
- [ ] Posting window confirmed and scheduled

## Promo Code and UTM Templates

### Promo Code Format

`CREATOR_[SHORTNAME]` (e.g. `CREATOR_JANE`)

### Landing URL Pattern

```
https://dfc.example.com/event/[eventslug]?promo=CREATOR_[SHORTNAME]&utm_source=creator&utm_medium=instagram&utm_campaign=[eventslug]&utm_content=[creator]
```

### UTM Examples

| Channel      | UTM                                                                                                              |
| ------------ | ---------------------------------------------------------------------------------------------------------------- |
| Cold ad      | `?utm_source=meta&utm_medium=cpc&utm_campaign=[eventslug]&utm_content=adA&utm_term=lookalike1`                   |
| Creator link | `?utm_source=creator&utm_medium=organic&utm_campaign=[eventslug]&utm_content=creator_jane&utm_term=CREATOR_JANE` |

### Tracking Instructions

- Record each promo code and landing URL in the campaign spreadsheet with creator handle, payment option, and expected post window.
- Use the promo code to attribute ticket sales and calculate revenue share or flat fee reconciliation.

## Poster and Video Specs

### Poster Specs

| Type              | Size         | Format                         | Notes                                           |
| ----------------- | ------------ | ------------------------------ | ----------------------------------------------- |
| Main social       | 1080×1350 px | JPG, sRGB, 85% quality         | `poster_1080x1350_[eventslug]_[creator].jpg`    |
| Stories/Vertical  | 1080×1920 px | JPG, sRGB, 85% quality         | `thumbnail_1080x1920_[eventslug]_[creator].jpg` |
| Print (if needed) | 3000×4500 px | TIFF or high-quality JPG, CMYK | —                                               |

### Video Specs

- **Aspect:** Vertical 9:16 (Reels / Shorts / TikTok)
- **Resolution:** 1080×1920
- **Codec:** H.264, AAC audio, 30 fps
- **Max file size:** 200 MB recommended
- **Captions:** Burn-in or SRT file included for accessibility

## Creator Delivery Window and Requirements

### Posting Window

Posts go live between **Day −7** and **Day −1** relative to event date. Stories may be posted Day −1.

### Mandatory Requirements

- Include disclosure text for paid posts: "Paid partnership with Data Fight Central" or `#ad`.
- Tag `@DataFightCentral` and include the landing page link with UTM.
- Provide screenshots of the live post and reach metrics within 72 hours of posting.
- If revenue share chosen, report promo code sales weekly until final reconciliation.
