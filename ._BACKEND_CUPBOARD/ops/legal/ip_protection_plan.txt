============================================================
 DATA FIGHT CENTRAL — IP PROTECTION PLAN
 Sole Trader / ABN Structure (Zero-Budget Path)
 Date: 28 March 2026
 Classification: CONFIDENTIAL
============================================================

NOTE: This plan is tailored for a sole trader operating under
an ABN (no Pty Ltd company). All IP is owned by the individual
until a company entity is formed. Adjust when structure changes.

============================================================
 1. BUSINESS REGISTRATION (DO FIRST — FREE/CHEAP)
============================================================

STEP 1: Register ABN (free)
  - Apply at abr.gov.au
  - Needed for invoices, Stripe, contracts, and IP filings
  - Takes ~10 minutes online

STEP 2: Register Business Name (if trading as "Data Fight Central")
  - Register at asic.gov.au — ~AUD $39 for 1 year or $92 for 3 years
  - Required if you trade under a name other than your legal name

STEP 3: Domain ownership
  - Ensure datafightcentral.com WHOIS is in your name
  - Buy protective domains (.au, .io, common misspellings) when budget allows

NOTE ON PTY LTD:
  When you can afford it (AUD $538 ASIC fee), form a Pty Ltd and
  transfer all IP via a deed of assignment. Until then, sole trader
  ABN is legally valid for all steps below.

============================================================
 2. TRADE MARKS (PRIORITY: HIGH — AUD $250 per class)
============================================================

STEP 1: Search first (free)
  - Search at search.ipaustralia.gov.au/trademarks
  - Also search Google, WHOIS, social media handles

STEP 2: File Australian trade mark application
  - Cost: AUD $250 per class (online filing)
  - Classes to consider:
    Class 9  : Software, apps, streaming platforms
    Class 41 : Entertainment, sporting events, streaming
    Class 42 : SaaS, platform services
  - Start with ONE class (most relevant) if budget is tight
  - Takes ~7-8 months for registration

MARKS TO PROTECT (priority order):
  1. DATA FIGHT CENTRAL / DFC (word mark)
  2. Logo mark (when finalised)
  3. CHUCKYA (if filing separately)
  4. Key taglines or event series names

BUDGET-FRIENDLY: File Class 41 first (entertainment + streaming),
add classes later.

 3. COPYRIGHT (FREE — AUTOMATIC)
============================================================

Copyright is automatic in Australia upon creation. No registration
required. However, EVIDENCE and RECORD-KEEPING matter:

DO NOW:

  [x] Commit code to private Git repo with timestamps
  [x] Keep dated backups of designs, videos, documents

IF DISPUTED: Git commit history, file metadata, and dated archives
serve as evidence of authorship and creation date.

============================================================
 4. PROVISIONAL PATENT (AUD ~$110)
============================================================

WHAT TO FILE: Provisional patent application with IP Australia
  - Cost: AUD $110 (online, standard fee)
  - Establishes priority date
  - 12 months to convert to full patent or PCT

WHAT TO INCLUDE:
  - Title and abstract
  - Background and problem statement
  - Detailed description (step-by-step operation)
  - Diagrams and sequence charts
  - Example implementations
  - Variants and alternatives
  - Inventor name and contact

SEE: ops/legal/provisional_patent_template.txt for full template

FILING STEPS:
  1. Fill template as plain-English PDF
  2. File at ipaustralia.gov.au (online)
  3. Pay AUD $110
  4. Within 12 months: engage patent attorney to convert

 7. ENFORCEMENT & MONITORING (FREE/CHEAP)
============================================================

  [ ] Monitor social media for brand misuse

   the intellectual property rights of Data Fight Central,

 9. WHEN YOU FORM PTY LTD (LATER)
============================================================

When you can afford it:
  1. Register Pty Ltd with ASIC ($538)
  2. Execute IP Assignment Deed — transfers all IP from you
     (sole trader) to the company
  3. Update all contracts to reference the company
  4. Update Stripe, Firebase, and vendor accounts
  5. Update trade mark and patent ownership records
