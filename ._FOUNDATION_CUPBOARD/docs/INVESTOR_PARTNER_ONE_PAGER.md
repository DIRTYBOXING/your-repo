# Apex Investor One-Pager

## Project
Data Fight Central (DFC) is a cloud-native combat-sports operating system that converts event and media inputs into canonical truth, ranked visibility, and monetizable experiences for promoters, creators, and fans.

## Elevator
DFC ingests events and media, normalizes canonical objects, ranks content with promotion controls, and publishes premium feeds and PPV funnels. The platform is production-ready at the spine level and can be productized into a polished consumer and operator experience quickly.

## Problem
Promoters and operators lack a unified operational layer for event orchestration, promotions, media evidence, and monetization. Current tooling is fragmented, manual, and not optimized for short-form discovery or PPV monetization.

## Solution
A single spine that supports:
- Source intake -> Normalize -> Rank -> Publish
- Feed and promotion seeding that the Flutter face already consumes
- PPV checkout and creator monetization flows
- Admin ops console for promotions, audits, and campaign control

## Traction and assets
- Cloud Run backend with health endpoints and deploy automation
- Firestore canonical collections for events, promotions, and feed items
- Flutter app that reads feed items and renders promotion content
- Seed scripts, promotions router, and CI smoke tests in repo
- Docs: spine, face build plan, architecture one-pager, pilot SOW

## Market opportunity
Niche verticals in combat sports, live events, and creator monetization with adjacent expansion into grassroots sports and community programs. Monetization via PPV revenue share, promoter SaaS, creator subscriptions, sponsorships, and merch.
