## 7. DATA SECURITY

7.1. **Encryption in Transit:** All data transmitted between your device and our servers is encrypted using TLS 1.2+ (TLS 1.3 preferred).

7.2. **Encryption at Rest:** Stored data is encrypted using AES-256 via Google Cloud's default encryption.

7.3. **Authentication Security:** Passwords are hashed using industry-standard algorithms (bcrypt via Firebase Auth). We never store plaintext passwords.

7.4. **Access Controls:** Role-based access control (RBAC) limits employee access to data on a need-to-know basis.

7.5. **Infrastructure:** Data is hosted on Google Cloud Platform infrastructure with SOC 2 Type II, ISO 27001, ISO 27017, and ISO 27018 certifications.

7.6. **Incident Response:** In the event of a data breach affecting your personal information, we will notify you within seventy-two (72) hours as required by GDPR Art. 33 and the Notifiable Data Breaches scheme under the Australian Privacy Act.

7.7. **Firestore Security Rules:** Granular document-level access controls enforce authentication, ownership, role-based permissions, field validation, and data immutability on audit records.
