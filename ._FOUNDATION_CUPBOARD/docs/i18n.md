# Internationalization

## Languages (Phase 1)

- English (en) — Primary, complete
- Hindi (hi) — India market
- Arabic (ar) — Middle East market (RTL support)
- Spanish (es) — Latin America + Spain
- Portuguese (pt-BR) — Brazil market

## Languages (Phase 2)

- Japanese (ja) — Japan market
- Thai (th) — Thailand (Muay Thai stronghold)
- Russian (ru) — Eastern Europe + Central Asia
- French (fr) — Francophone Africa + France
- Tagalog (tl) — Philippines market

## Features

- **Localized pricing** — Currency auto-detection, regional pricing tiers
- **Localized events** — Event listings filtered by region/timezone
- **Localized content** — Feed content prioritized by language preference
- **RTL support** — Full right-to-left layout for Arabic, Hebrew
- **Date/time formatting** — Locale-aware (DD/MM/YYYY, AM/PM vs 24h)
- **Weight/measurement** — kg/lbs toggle, metric/imperial per region

## Implementation

- Flutter `intl` package + ARB files in `locales/`
- Firestore content: `locale` field on posts, events, promotions
- Fallback chain: user locale → region default → English
- Translation workflow: source strings → professional translation → QA → deploy
