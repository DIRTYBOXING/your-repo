# DFC Promotion Rules

**Version:** 1.0
**Effective:** 2026-03-27

## Rules (Short Version)

1. **Respectful promotion only:** no hate speech, harassment, or trolling in comments or promoted posts.
2. **No doctored or misleading assets:** all posters must accurately reflect event details and fighters.
3. **Copyright & trademarks:** promoter must supply rights for logos, fighter photos, and broadcast marks.
4. **Age restrictions:** all PPV listings must include age rating (18+ where required).
5. **Refunds & terms:** promoter must provide refund policy and ticketing terms; DFC will display these in the purchase flow.
6. **Reporting:** DFC reserves the right to remove content that violates rules; promoter will be notified and must respond within 48 hours.

## Legal Compliance

- Only publish images you have rights to. If a promoter supplied a poster or the promoter has already publicly promoted the poster, record the promoter's written permission and model releases before using fighter photos or trademarked logos.
- **Typographic fallback:** If permission is not yet obtained, publish the typographic poster generated by the script (safe to publish).
- **Record licenses and releases** in `docs/legal/` and add a `license` block to each event JSON.
- **Do not use broadcast screenshots, copyrighted logos, or trademarked artwork** without explicit written permission. Replace logos with plain text if permission is not granted.
- **Retention:** keep signed PDFs and emails in `docs/legal/` and reference them in the event JSON `license.contact`.
