# 🧪 MULTI-PERSONA UX TESTING

Test DFC as 8 different people:

## 👩‍🦰 Female Fighter

- Profile creation, weight class logic, training tools
- ATLAS insights, fight history, safety tools, social feed
- Check for: missing divisions, UI bias, incorrect ranking logic

## 🧔 Male Fighter

- Rankings, fight history, training plans, injury tracking, event registration
- Check for: duplicate fighters, incorrect weight classes, missing amateur divisions

## 🥋 Coach / Trainer

- Manage fighters, create training plans, upload sparring footage
- ATLAS analysis, gym dashboard
- Check for: missing permissions, no bulk import, no analytics

## 🏋️‍♂️ Gym Owner

- Gym profile, add fighters/coaches, event hosting, membership tools
- Check for: no verification, no revenue dashboard, no scheduling

## 🎤 Promoter

- Create fight cards, add fighters, PPV setup, ticketing, event analytics
- Check for: no matchmaking AI, no automated posters, no promoter dashboard

## 👀 Fan / Viewer

- Rankings, news, fighter profiles, events, social feed
- Check for: slow loading, missing filters, no personalization
