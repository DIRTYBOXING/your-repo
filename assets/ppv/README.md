# PPV Poster Assets

Generated by `tools/generate_ppv_posters_gritty.sh` or `tools/generate_ppv_posters_gritty_node.js`.

## Naming convention

- `{eventId}_hero.jpg` — 1600×2400 hero poster
- `{eventId}_thumb.jpg` — 1200×1200 square thumbnail
- `{eventId}_banner.jpg` — 1920×600 banner
- `{eventId}_portrait.jpg` — 1080×1920 story/portrait
- `{eventId}_preview.jpg` — 640×360 preview card
- `{eventId}_photo_left.jpg` — Left fighter photo (optional input)
- `{eventId}_photo_right.jpg` — Right fighter photo (optional input)
- `grunge_texture.png` — Shared grunge overlay (optional)

## Generation

```bash
# Single event
./tools/generate_ppv_posters_gritty.sh data/events/ppv-ufc-327.json

# All events
./tools/generate_ppv_posters_gritty.sh --all data/events/

# Node.js variant (WebP output)
node tools/generate_ppv_posters_gritty_node.js --all data/events/
```
