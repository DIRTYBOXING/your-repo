# Fighter Data

Store fighter metadata JSON files here.

## Naming convention

`<fighter-slug>.json` — e.g. `haze-hepi.json`, `jack-della-maddalena.json`

## Schema

```json
{
  "fighterId": "haze-hepi",
  "name": "Haze Hepi",
  "nickname": "The Huntsman",
  "record": "3-1",
  "weightClass": "Heavyweight",
  "sport": "BKFC",
  "country": "NZ",
  "photoUrl": null,
  "events": ["ppv-bkfc-townsville-hepi"]
}
```
