# 5-Minute Demo Script for Apex Meeting

## 0:00-0:30 - Quick intro
- One sentence mission and one sentence traction.
- State the ask succinctly.

## 0:30-1:30 - Live staging walkthrough start
- Show Home feed with a seeded promotion visible.
- Highlight how promotions are created and how they surface immediately.

## 1:30-2:30 - Event and PPV flow
- Open an event page.
- Show media playback.
- Demonstrate the PPV checkout flow to completion using test Stripe mode.
- Emphasize conversion path and receipts.

## 2:30-3:30 - Admin ops and promotions control
- Switch to `/ops/promotions`.
- Create a short-lived promo.
- Show it writing to Firestore and appearing in the feed.
- Show audit log entry.

## 3:30-4:30 - Technical backbone and pilot plan
- Briefly show architecture one-pager: Cloud Run, Firestore, Cloud Storage, Secret Manager, CI smoke tests.
- Explain GPU pilot objective and expected delta.

## 4:30-5:00 - Close and ask
- Restate the ask: capital, introductions, and a 2 to 4 week co-engineering pilot.
- Propose next step: 20-minute technical deep dive with SEs and a signed pilot SOW.

## Quick meeting pitch bullets
- One line: DFC is the operational backbone for combat sports - feeds, promotions, PPV, and operator control in one platform.
- Why it matters: Promoters need a single system to run events, monetize, and scale creator supply.
- What we have: Production spine, Flutter face, seed scripts, promotions router, Stripe direction, CI smoke tests.
- What we need: $1M to $3M, partner intros, and a 2 to 4 week co-engineering GPU pilot.
- Immediate milestone: Premium face MVP, 3 promoter pilots, GPU benchmark with partner within 90 days.
