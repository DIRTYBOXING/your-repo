# DFC Premium UI Roadmap

## Phase 1: Core Shell

- Rename the primary lanes to match user intent: Home, Watch, Explore, Network, Profile.
- Keep one shared shell backdrop and accent logic across tabs so every surface feels related.
- Standardize top-surface spacing so the shell bar, intro blocks, and first content rail align.

## Phase 2: Media System

- Route posters, cover art, OG preview images, and event thumbnails through one shared media frame.
- Use one fallback policy: branded loading state, safe image sanitization, consistent rounded corners, and readable bottom scrims.
- Keep generated fight-card art as the fallback behind remote posters instead of dropping to flat placeholders.

## Phase 3: Home

- Treat Home as the operating rail for the product, not just a mixed feed.
- Keep the order: live pulse, stories/live channels, compose, trending, ranked feed.
- Promote clearer lane filters: live, events, community, news.

## Phase 4: Watch

- Treat Watch as the premium commerce and viewing lane.
- Keep hero art cinematic, simplify the tab labels, and keep the schedule/library language consistent.
- Standardize event posters so live, upcoming, and purchased PPV surfaces share the same card logic.

## Phase 5: Network

- Position Network as relationship management for fighters, gyms, promoters, fans, and media.
- Keep suggestion cards flatter and more information-first than consumer-social cards.
- Use the same card, badge, and media rules as Home and Watch.

## Phase 6: Refinement

- Replace isolated one-off image widgets over time with the shared media frame.
- Unify loading, empty, and error states across the major tabs.
- Use motion only where it communicates live state, ranking shifts, or user progress.
