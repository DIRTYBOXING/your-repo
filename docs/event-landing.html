Conference Agenda

Day 1: Athlete Safety & Biometric Processing

09:00 - 10:00 AEST

Opening Keynote: Re-imagining Safety via Data Integration

An overview of how Data Fight Central uses high-frequency hardware queues to support real-time wellness evaluation, and how open-source collaboration saves lives.

10:30 - 12:00 AEST

Workshop: Live Heart-Rate Strain Verification Loops

Hands-on trace showing sensor streams entering Firebase functions, processing, and triggering emergency protective indicators securely.

13:30 - 15:00 AEST

Panel: The Ethics of Athlete Performance Bio-Tracking

Privacy professionals and sports physicians clarify safe storage buffers and data-minimization practices for wearables.

Day 2: AI & Streaming Engines at Scale

09:30 - 11:00 AEST

Deploying Gen2 Webhooks with Real-Time Latency

An in-depth guide to modernizing backend webhooks. Deep dive into building region-optimized HTTP listeners capable of handling 50k requests/min.

11:30 - 13:00 AEST

NVIDIA & Google Ecosystem Integration

Leveraging the NVIDIA Inception toolkit with Cloud Run to generate sub-second fight prediction matrices and video indexing.

14:30 - 16:00 AEST

Open-Source Release and Q&A

Showcasing DFC's upcoming open-source community utilities and outlining paths for external community developers to contribute.
