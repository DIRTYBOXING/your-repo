# DFC Grant Evidence Room

Central repository for grant-review evidence artifacts.

## Purpose

Provide fast, auditable proof of DFC execution quality, governance discipline, and measurable outcomes.

## Contents

- `EVIDENCE_CHECKLIST.md` — required artifact checklist per submission
- `KPI_SNAPSHOT_TEMPLATE.md` — reusable KPI snapshot format
- Sponsor and launch materials from `docs/launch/`
- Architecture/governance references from `docs/`

## Update cadence

- Weekly: KPI and release evidence refresh
- Monthly: grant narrative alignment check
- Pre-submit (T-7 days): checklist completeness review
