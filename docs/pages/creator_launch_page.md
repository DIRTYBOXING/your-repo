# Creator Launch Page

## Purpose
Onboard creators, publish content, run Week-1 launch, promo codes, payouts, and creator support.

## Onboarding Checklist
- [ ] Creator signs up and connects Stripe Connect
- [ ] Upload hero clip and 3 premium clips
- [ ] Set subscription price and PPV price
- [ ] Provide payout details and tax info

## Launch Week Plan
- Day 0: Publish hero clip, enable promo code `DFC{handle}30`
- Day 1: Live mini-class, collect emails
- Day 3: Paid ad push with top clip
- Day 7: Review metrics and adjust pricing

## Creator Toolkit
- Promo code generator
- Analytics: subscribers, conversion, ARPU
- Payout request flow via Stripe Connect batch payouts
