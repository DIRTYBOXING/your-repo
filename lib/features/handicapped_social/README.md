# Handicapped Social Workflow Module

This module provides a simplified, accessible social workflow for users needing easy navigation, large UI elements, and clear guidance.

## Features

- Inbox: Large text, simple refresh, clear sender info
- Outbox: Large text, simple refresh, clear receiver info
- Friend/Group Management: Easy add button, visible friend list
- Posting/Reading: Simple create post, large feedback, image upload

## Accessibility

- High contrast colors for visibility
- Large buttons and text for easy tapping
- Minimal navigation steps
- Tooltips and help icons for guidance
- Error messages are clear and actionable

## Usage

1. Open the Inbox screen to view received messages. Use the search bar to filter messages. Tap refresh for latest.
2. Open the Outbox screen to view sent messages. Use the search bar to filter. Tap refresh for latest.
3. Manage friends/groups with the Friend Group screen. Tap 'Add Friend' for easy addition.
4. Create and read posts in the Post Read screen. Use large buttons for posting and feedback.

## Tips for Users with Disabilities

- Use the search bar for quick filtering.
- Tap help icons for tooltips and guidance.
- If you get lost, return to the main tab bar or home screen.
- All screens are designed for minimal steps and maximum clarity.

## Troubleshooting

- If you see an error, check your internet connection and try again.
- If buttons are hard to tap, increase device accessibility settings.
- For further help, contact support or use the onboarding overlay for walkthrough.

## Integration

Import and use the screens in your app for accessible social workflows. All backend logic is wired for Firestore and scalable for advanced features.
